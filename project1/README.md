В данной директории хранятся следующие файлы:
1. Используемые датасеты: информация об аптеках Москвы (pharmacies.csv), о больницах Москвы (medical_institutions.csv) и о станциях метро Москвы (subway.csv);
2. Основной код проекта (project1.ipynb) с выводами по пунктам 1 и 4 (см. README репозитории);
3. Результаты, полученные в рамках выполнения данного проекта: интерактивная карта Москвы с нанесенными на нее топ-5 самых популярных сетей аптек (top5_chains_map.html), доступность аптек относительно станции метро (pharmacies_accessibility.csv), доступность больниц относительно станции метро (pharmacies_accessibility2.csv).
