# data-engineer-projects
Данная репозитория содержит пет-проекты по инженерии данных. Каждый проект будет находиться в отдельной поддиректории и содержать основной код программы, а также используемые датасеты и полученные результаты/выводы.

Данный README будет содержать краткое описание поставленной задачи в рамках каждого проекта, а также используемые технологии для его написания.

## Проект №1

В этом проекте используется датасет, выгруженный с помощью API 2гиса и содержащий данные по аптекам и сетям аптек (название, адрес в координатах, обычный адрес, ID и тип аптеки (сеть аптек или нет)).

Были поставлены три основные задачи: 
1. Выяснить, какие сети аптек наиболее распространены по Москве (т.е. выявить топ-5), а также нанести их на интерактивную карту Москвы;
2. Выяснить самые доступные аптеки относительно каждой станции метро в радиусе 800 метров от метро (с указанием адреса аптеки, расстояния от нее до метро в метрах и минутах), а также общее количество попавших в данный радиус аптек;
3. Выяснить самые доступные больницы относительно каждой станции метро в радиусе 800 метров от метро (с указанием адреса больницы), а также общее количество попавших в данный радиус больниц;
4. Доказать гипотезу о том, что среднее число аптек возле больниц больше, чем возле случайных точек на карте (количество случайных точек равно количеству рассматриваемых больниц).

Используемые технологии: PostgreSQL, Python (pandas, folium, polars), Jupyter Notebook.
